Класс Year
----------
Для создания объектов "год" и массива этих объктов.

Year.makeYearArr(period, initialSpareFund) для создания массива объектов "год"
	period - на сколько лет рассчитываем (количество объектов "год")
	initialSpareFund - начальный капитал
	
Year.data - результат работы Year.makeYearArr(), массив объектов.

Класс Data
----------
Для создания объектов с методами обработки массивов объектов, 
возвращающими массивы (массивы массивов) для построения графиков
